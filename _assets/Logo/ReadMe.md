# PWA Icons

Design Logo and use Logo in Icons and Splash to generate all sizes

In Art boards, Formats should have scales of and suffixes of:

scale          suffix      Format
72  px         72x72       png
96  px         96x96       png
128 px         128x128     png
144 px         144x144     png
152 px         152x152     png
192 px         192x192     png
384 px         384x384     png
512 px         512x512     png


export all sized to root `/public/images/icons`
